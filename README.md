# Pullit

Pullit is a real-time credential finder. 

### Installation

- ``` git pull https://github.com/filtration/pullit.git ```
- ``` sudo chmod +x install.sh  ```
- ``` . ./install.sh ```
- ``` python ./pullit.py  ```
